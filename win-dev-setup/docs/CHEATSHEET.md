# 快捷键速查

## WezTerm

### 窗口 / 全屏
| 快捷键 | 功能 |
|---|---|
| `Ctrl+Enter` | 切换全屏 |
| `Alt+n` | 新建窗口 |
| `Ctrl+Shift+P` | 命令面板 |
| `Ctrl+Shift+F` | 搜索终端内容 |

### 标签页
| 快捷键 | 功能 |
|---|---|
| `Alt+t` | 新建标签 |
| `Alt+m` | 标签导航器 |
| `Alt+w` | 关闭当前 |
| `Alt+Shift+L` / `Alt+Shift+H` | 切换下/上标签 |

### 分屏
| 快捷键 | 功能 |
|---|---|
| `Alt+d` | 水平分屏 |
| `Alt+Shift+D` | 垂直分屏 |
| `Alt+h/j/k/l` | 焦点 左/下/上/右 |
| `Alt+方向键` | 调整分屏大小（每次 5） |

### 剪贴板 / 鼠标
| 操作 | 功能 |
|---|---|
| **左键拖动选中** | 松开自动复制（选中即复制）|
| `Ctrl+C` | 有选区 → 复制；无选区 → 中断进程 |
| `Ctrl+V` | 剪贴板有图 → 保存 PNG 并粘贴路径；否则粘贴文本 |
| **右键** | 粘贴 |
| `Ctrl+左键` | 打开链接 |

## Yazi

### 移动 / 打开
| 按键 | 动作 |
|---|---|
| `hjkl` 或方向键 | 左/下/上/右 |
| `Enter` | 进目录 或用 nvim 打开文件 |
| `q` | 退出 |
| `Q` | 退出并把 cwd 传回父 shell（用 `y` 函数调用时自动生效） |
| `!` | 在当前目录开一个 shell |
| `~` / `?` | 查看所有快捷键 |

### 搜索
| 按键 | 动作 |
|---|---|
| `/` | 搜索当前目录文件名 |
| `n` / `N` | 下/上一个匹配 |
| `s` | 按内容搜（ripgrep） |
| `z` | fzf 模糊跳（需 zoxide） |

### 文件操作
| 按键 | 动作 |
|---|---|
| `y` | 复制 |
| `x` | 剪切 |
| `p` / `P` | 粘贴 / 强制粘贴（覆盖） |
| `d` / `D` | 删除（回收站/永久） |
| `a` | 新建（末尾加 `/` 新建目录） |
| `r` | 重命名 |
| `o` | 用系统默认程序打开 |

## Neovim（HUAHUANVIM / Six 分支）

**Leader = 空格**，插入模式下按 `jk` 代替 Esc 回 Normal。

### 模式
| 按键 | 动作 |
|---|---|
| `i` | 插入模式 |
| `Esc` 或 `jk`（插入里） | 回 Normal |
| `v` / `V` / `Ctrl+v` | Visual / 行 Visual / 块 Visual |

### 文件与 buffer
| 快捷键 | 功能 |
|---|---|
| `<Space>e` | 打开/关闭左侧文件树 |
| `<Space>a` | 大纲（函数列表） |
| `<Space>sv` / `<Space>sh` | 垂直/水平分屏 |
| `<Space>x` | 关闭当前 buffer |
| `Shift+L` / `Shift+H` | 下/上一个 buffer |
| `<Space>nh` | 清除搜索高亮 |

### 编辑
| 快捷键 | 功能 |
|---|---|
| `yy` + `p` | 复制/粘贴行 |
| `dd` | 剪切/删行 |
| `u` / `Ctrl+r` | 撤销 / 重做 |
| `gcc`（Normal）/ `gc`（Visual） | 注释 |
| Visual 模式 `J`/`K` | 上下移动选中行 |
| `ciw` / `ci"` | 改词 / 改引号内容 |

### 搜索 / 跳转
| 按键 | 功能 |
|---|---|
| `/` | 向下搜 |
| `?` | 向上搜 |
| `n` / `N` | 下/上一个匹配 |
| `*` | 搜光标下的词 |
| `gd` | 跳到定义（LSP） |
| `K` | 悬浮文档（LSP） |
| `:Mason` | 管理 LSP 服务器 |
| `:Lazy` | 管理插件 |

## PowerShell / zsh 自定义命令

| 命令 | 作用 |
|---|---|
| `y` | yazi 增强版（退出时自动 cd） |
| `ll` / `la` / `lt` | eza 详细 / 含隐藏 / 树状 |
| `gs` / `gd` / `gl` | git status / diff / log |
| `..` / `...` | cd 上/上上级 |
| `Enable-Proxy` / `Disable-Proxy` (Win) | 切换代理 |
| `proxy_on` / `proxy_off` (Mac) | 切换代理 |
